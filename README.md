Project Management Dashboard – Power BI
Overview

This Power BI dashboard provides an end-to-end view of project performance, covering KPIs, timelines, regional insights, project phases, and detailed status tracking. It helps stakeholders monitor project progress, cost-benefit ratios, timelines, and completion metrics in an interactive format.

Key Features:

Project Summary Page
Displays total projects, total cost, total benefits, and benefit–cost ratio.
Includes insights by region, department, month, project status, and phase using cards, column charts, doughnut charts, and line charts.

Project Timeline Page
Gantt chart showing start date, end date, and project duration.
Line charts displaying total projects by month or year with completion percentage.

Project Detail Page
Interactive table with conditional formatting to highlight completed, on-hold, delayed, or canceled projects.

Power BI Navigation
Includes custom page navigation buttons designed for a seamless user experience.

Power Query Data Cleaning
Raw project data is cleaned and transformed for accurate reporting.
Data types corrected, null values removed, and new calculated columns created.

DAX Measures and Date Table
A dynamic date table is created using DAX.
Measures built for KPIs, timelines, and performance comparison.

Custom Visual Design
PowerPoint backgrounds used to enhance the visual appearance of the report.
